PASSOS A PASSOS 

1- Baixar projeto 

git clone https://github.com/qrpague/qrpague-server.git

2- Pre requisitos para instalação

instalar nodejs version > 8 
instalar mongodb version > 3.2 
acesso a Internet

3- config file perfil network 
 
 abrir arquivo config.js  na raiz do projeto e definir de acordo com sua infra-estrutura as devidas configurações.
 
3- Comandos para instalar

sudo npm install pm2 -g 

npm install

sh start-server.sh qrpague-server
